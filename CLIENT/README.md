# Client Assets

This folder is reserved for client-side artifacts related to the `0.59.x` compatibility line.

## Included

- `patches/omr-tracker`
- `patches/post-tracking`

These are patched versions of the client tracker scripts used by `v0.59.2-3KTEST` images.

## Notes

- Keep large image files outside git when possible (or publish them as release assets).
- Use `Patch.md` in the repository root for manual patch instructions on already deployed clients.
